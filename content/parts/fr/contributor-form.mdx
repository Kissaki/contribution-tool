## Soyez cité dans votre contribution !

Open Terms Archive est une initiative Open Source qui s'appuie sur des contributeurs comme vous.

Nous voulons donc vous donner **tout le crédit** pour votre contribution et pour ce faire, nous avons besoin d'une information de votre part.

2 possibilités s'offrent à vous :

### Vous voulez être cité

Super ! Veuillez saisir l'une des informations suivantes dans le champ ci-dessous :

- votre adresse e-mail GitHub → vous serez <a href="https://docs.github.com/en/pull-requests/committing-changes-to-your-project/creating-and-editing-commits/creating-a-commit-with-multiple-authors" target="_blank">listé comme co-auteur</a>
- une adresse e-mail → celle-ci apparaîtra sur le changement proposé sur GitHub
